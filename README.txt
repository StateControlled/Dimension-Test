Developed with NeoForge for Minecraft 1.20.4

This mod is a proof of concept that aims to create a Backrooms-like dimension.
Specifically, the objective to to create Poolrooms-type terrain.
The ultimate goal is to create a backrooms-like dimension using either an infinitely generated jigsaw structure or use a custom carver (extending the CaveWorldCarver) to randomly generate underground terrain.
